# 🚀 Portfolio Website – Interactive Developer Portfolio

Un portfolio moderne et interactif développé en **HTML, CSS et JavaScript vanilla**, avec animations avancées, curseur personnalisé, fond dynamique et formulaire de contact fonctionnel via EmailJS.

---

## ✨ Aperçu

Ce projet est un site portfolio conçu pour mettre en valeur des compétences en développement web à travers une expérience utilisateur immersive :

- 🎯 Curseur personnalisé animé
- 🌌 Fond interactif avec particules et code flottant
- 📱 Design responsive
- ✨ Animations au scroll
- 📩 Formulaire de contact fonctionnel (EmailJS)
- ⚡ Interface fluide et moderne

---

## 🛠️ Technologies utilisées

- HTML5
- CSS3 (animations & responsive design)
- JavaScript (ES6+)
- Canvas API
- EmailJS
- Intersection Observer API

---

## 🎨 Fonctionnalités principales

### 🖱️ Curseur personnalisé
- Curseur dynamique avec effet de suivi
- Effets hover sur liens, boutons et tags
- Désactivation automatique sur mobile

### 🌌 Background animé
- Particules interactives
- Lignes de connexion dynamiques
- Code flottant stylisé (style dev environment)
- Performance optimisée avec Canvas

### 📱 Navigation & UX
- Navbar qui change au scroll
- Sections animées au scroll (scroll reveal)
- Expérience fluide et moderne

### 📂 Projets
- Affichage progressif des projets
- Bouton "Voir plus"

### 📩 Formulaire de contact
- Envoi d’emails via EmailJS
- États de chargement, succès et erreur
- Reset automatique du formulaire
